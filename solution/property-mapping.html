# 属性映射

本节内容施工中...
属性映射的信息在生成预制件的过程中收集（组件信息、业务信息、属性、编码值）
属性映射具体思路就是，通过选择器定位原子，然后与标识建立联系。这个联系电子化就是属性映射表。
比如“XXX业务 XXX产品 一级标题组件 TEXT[fontSize = 某标识]” 就可以让指定产品线下所有一级标题字号以某标识生效。
选择器的粒度是自由的，也可以是无关业务的定义“一级标题组件 TEXT[fontSize = 某标识]”，也可以具体到某个页面的实例。
属性映射的选择器不是粗暴地通过一个CRUD完成的，选择器的各部分是在新建、管理设计资产的日常操作中建立的，对于使用者而言只是在修改组件属性，只是一个视图操作。
